Design of Gurbani Vocab


Technical Design

Gurbani Vocab is a three-tiered web application.

User Interface Layer
Jinja2 is the template language that allows Python code to be written and accessed from the user interface. The user interface itself is coded using a combination of HTML5 and CSS.

Logic Layer
The primary logic is written using Python3. The web framework used to build the web application is Flask. It handles HTTP requests, session management, etc.

Data Layer
The database - used to store and analyze data - is MongoDB which is accessed via Python using PyMongo. Sensitive data such as passwords are encrypted in Python using Passlib - a password hashing library used in Python versions 2 and 3.

Functionality

Gurbani Vocab utilizes a list of the 330 most common words from the Guru Granth Sahib (Sikh scripture) that have been divided into word lists of 15 for user convenience.

Study
The Study mode allows the user to view the entire list of words, definitions, and transliterations to study.

Quiz
The Quiz mode asks the user multiple choice questions about each word to test the user understanding of the definitions. The user's progress is stored in the database.

Progress
User progress can be retrieved in either the Progress or Profile tab. The user can view the percent accuracy as well as all of the words they have answered incorrectly and correctly for each vocabulary list.

Classes
The user also has the option to enroll in a class using a class code assigned by a teacher user. This allows the teacher to monitor the user's progress for each vocabulary list.
